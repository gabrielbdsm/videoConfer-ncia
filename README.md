## Instalação

1. Clone este repositório:

    ```bash
    git clone https://github.com/gabrielbdsm/videoConferencia.git
    ```

2. Instale as dependências:

    ```bash
    npm install
    ```

3. Execute o aplicativo:

    ```bash
    npm rum start-dev
